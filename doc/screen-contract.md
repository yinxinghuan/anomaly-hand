# 《异常手牌》屏幕与状态合同

## 1. 行动员选择

- 玩家问题：我是谁，我的能力有什么不同？
- 主焦点：1 张大幅收藏级英雄档案卡，完整展示人物、被动与签名技。
- 主动作：选择一位行动员。
- 次动作：查看“战斗规则”短说明。
- 持久信息：游戏名、档案编号、每次进入随机发出的 3 张行动员档案。
- 临时反馈：点击随机档案卡后大卡即时换人，编号、人物、被动与签名技同步更新。
- 视觉承诺：大卡必须是完整档案物件，含专色装订边、环形定位轨、角标与能力信息带；索引只作为低优先级档案抽屉，不能与主卡争夺注意力。
- 恢复：任何时候可改选；确认后进入“活体档案”说明页。

## 2. 活体档案说明

- 玩家问题：这局为什么会持续成长，何时结束？
- 主焦点：行动员肖像、个人档案、对手封存、异变节点三条编号协议。
- 主动作：点击“进入第一遭遇”。该页不自动消失。
- 持久信息：个人卡生成状态；生成不可阻断继续战斗。
- 视觉承诺：这是档案系统的标题页，不是轻量提示或黑色普通弹窗。

## 3. 遭遇导入

- 玩家问题：面对谁，对方下一步做什么？
- 主焦点：先用全屏章节牌明确敌人姓名与身份，再让敌方档案、行动员档案依次进入战斗桌。
- 主动作：等待双方档案部署完成；只在“你的回合”章节收束后选择手牌。
- 持久信息：轮次、轮内遭遇、双方生命、英雄姓名。
- 临时反馈：`敌对档案已识别` 章节牌显示敌人姓名与身份；约 1,730 ms 后敌方档案从左侧飞入，约 2,550 ms 后行动员档案从右侧飞入；双方标签明确标记“敌人 / 你”。
- 输入合同：导入的所有阶段显示“暂不可操作”，三张手牌具有 `disabled` 属性和容器级 `pointer-events: none`，键盘 `1/2/3` 也不生效。约 4,700 ms 后才进入可操作的行动员回合。
- 时长：标题 1,650 ms；敌方档案入场 760 ms；行动员档案入场 780 ms；“你的回合”说明 1,250 ms。不得通过点击跳过。
- 视觉承诺：章节牌是全屏识别封面，使用专色封条、对角轨线、双层菱形定位章和大标题；不是短暂闪烁的一行提示。

## 4. 核心战斗

- 玩家问题：面对这个意图，我该打哪一张？
- 主焦点：敌对行动员档案、公开意图和 3 张带系列化版画插图的战术牌。
- 主动作：打出 1 张牌。
- 次动作：规则说明、静音。
- 持久信息：双方生命/格挡、遭遇进度、3 点序列。
- 临时反馈：伤害、格挡、暴露、英雄被动、签名技就绪。
- 视觉承诺：常态保持低频定位点与纸张纹理；命中瞬间才触发纸屑、套印射线、封条抖动和档案框错位。结算固定分两拍：先居中说明“攻击命中 / 格挡部署 / 敌方蓄力”等效果，再在约 680 ms 后放大显示“对象 + 数值”，评分与战术分退为辅助信息。
- 锁定：出牌后 420–900 毫秒内其他牌禁用。
- 恢复：若音频失败继续游戏；若图片加载失败显示姓名首字与几何剪影。

## 5. 战后强化

- 玩家问题：我想强化哪一条战术？
- 主焦点：3 张纵向收藏级改造卡；每张拥有独立语义专色、几何徽记、强化类型、大数值与效果说明。
- 主动作：选择一项。
- 持久信息：当前生命、下一遭遇编号。
- 临时反馈：按下即锁定；选中项恢复高饱和专色并盖“改造已装订”章，其他两项褪为低饱和炭纸，620 毫秒后切入下一遭遇。
- 视觉承诺：人物水印、改造档案编号和三张高密度收藏卡共同构成奖励峰值；每张卡都必须保留语义徽记、数值、名称和阅读区。
- 恢复：选择前可返回阅读当前英雄能力，不能退回已结束战斗。

## 6. 循环 / 失败

- 玩家问题：结果是什么，接下来做什么？
- 主焦点：第 7 场后的轮次封存章，或死亡时的英雄档案与结果章。
- 主动作：再次行动。
- 次动作：更换行动员。
- 持久信息：总分、轮次、封存遭遇数、最高连段、签名技次数。
- 临时反馈：轮次封存黄铜章节停留 2,400 ms 后继续；失败朱红断档线后提交一次分数。
- 视觉承诺：结案页沿用英雄档案卡和最终归档标，不用通用弹窗；结果章、统计和后续动作由同一张档案结构收束。
- 恢复：死亡页可重开、换人，平台内还可打开同一 UUID 的档案榜；榜单行可点击他人头像与姓名进入资料页。

## 6. 规则覆盖层

- 内容：敌人先公开意图；每回合选 1 张牌；不同类别连续出牌积累序列；序列 3 点出现签名牌。
- 打开方式：带文字的“规则”按钮，不使用问号 Emoji。
- 关闭方式：关闭 SVG、点击遮罩、键盘 Escape。
- 响应式：内容超过短屏时内部滚动；关闭按钮固定且不少于 44 × 44。

## 7. 加载与错误

- 英雄图加载中：显示卡框、姓名与低对比网点剪影，布局不跳动。
- 英雄图失败：显示姓名首字/物种符号形成的几何占位，不显示空圆。
- 全局异常：保留返回行动员选择的主按钮，并显示简短错误说明。
- 离线：玩法与本地资产可继续；平台查询与分享不是垂直切片依赖。
