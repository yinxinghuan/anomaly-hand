# 《异常手牌》屏幕与状态合同

## 1. 行动员选择

- 玩家问题：我是谁，我的能力有什么不同？
- 主焦点：1 张大幅收藏级英雄档案卡，完整展示人物、被动与签名技。
- 主动作：选择一位行动员。
- 次动作：查看“战斗规则”短说明。
- 持久信息：游戏名、档案编号、8 人横向档案索引。
- 临时反馈：点击索引后大卡即时换人，编号、人物、被动与签名技同步更新。
- 恢复：任何时候可改选；确认后进入第一场。

## 2. 遭遇导入

- 玩家问题：面对谁，对方下一步做什么？
- 主焦点：高细节敌方档案位图与第一条意图。
- 主动作：触碰任意手牌进入战斗。
- 持久信息：遭遇 `1/3`、双方生命、英雄姓名。
- 临时反馈：敌人档案盖入，与英雄档案形成前后压叠；意图图标从档案边缘滑出。
- 时长：700–900 毫秒；玩家按牌可提前结束导入。

## 3. 核心战斗

- 玩家问题：面对这个意图，我该打哪一张？
- 主焦点：敌方位图档案、公开意图和 3 张带印刷底纹的战术牌。
- 主动作：打出 1 张牌。
- 次动作：规则说明、静音。
- 持久信息：双方生命/格挡、遭遇进度、3 点序列。
- 临时反馈：伤害、格挡、暴露、英雄被动、签名技就绪。
- 锁定：出牌后 420–900 毫秒内其他牌禁用。
- 恢复：若音频失败继续游戏；若图片加载失败显示姓名首字与几何剪影。

## 4. 战后强化

- 玩家问题：我想强化哪一条战术？
- 主焦点：3 张纵向收藏级改造卡；每张拥有独立语义专色、几何徽记、强化类型、大数值与效果说明。
- 主动作：选择一项。
- 持久信息：当前生命、下一遭遇编号。
- 临时反馈：按下即锁定；选中项恢复高饱和专色并盖“改造已装订”章，其他两项褪为低饱和炭纸，620 毫秒后切入下一遭遇。
- 恢复：选择前可返回阅读当前英雄能力，不能退回已结束战斗。

## 5. 胜利 / 失败

- 玩家问题：结果是什么，接下来做什么？
- 主焦点：英雄档案与结果章。
- 主动作：再次行动。
- 次动作：更换行动员。
- 持久信息：总回合、签名技次数、强化列表、到达遭遇。
- 临时反馈：胜利青蓝封存线；失败朱红断档线。
- 恢复：两次点击以内开始新局。

## 6. 规则覆盖层

- 内容：敌人先公开意图；每回合选 1 张牌；不同类别连续出牌积累序列；序列 3 点出现签名牌。
- 打开方式：带文字的“规则”按钮，不使用问号 Emoji。
- 关闭方式：关闭 SVG、点击遮罩、键盘 Escape。
- 响应式：内容超过短屏时内部滚动；关闭按钮固定且不少于 44 × 44。

## 7. 加载与错误

- 英雄图加载中：显示卡框、姓名与低对比网点剪影，布局不跳动。
- 英雄图失败：显示姓名首字/物种符号形成的几何占位，不显示空圆。
- 全局异常：保留返回行动员选择的主按钮，并显示简短错误说明。
- 离线：玩法与本地资产可继续；平台查询与分享不是垂直切片依赖。
