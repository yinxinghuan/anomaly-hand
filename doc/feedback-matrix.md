## Timing

- 输入确认：同帧，视觉按下不等待逻辑。
- 翻牌预备：0–90 ms。
- 接触峰值：约 210 ms。
- 效果拍：命中后约 210–890 ms；先阅读“发生了什么”。
- 数值拍：约 890–2,230 ms；大号对象标签和数值完整停留，随后才进入下一阶段。
- 章节可读性：进入约 240 ms、完整停留、退出约 260 ms。章节内容不再由固定 860 ms CSS 动画提前变透明，而是严格跟随状态时长。
- 开局 / 下一遭遇：敌方识别标题 1,650 ms，随后敌方入场、行动员入场和玩家回合标题依次完成；从点击“接入档案”到输入解锁约 4,700 ms。敌方回应：1,350 ms；玩家回合：1,250 ms；强化：1,650 ms；胜负：1,850 ms。
- 玩家结算：出牌约 210 ms 后出现；效果拍保持约 680 ms，数值拍保持约 1,340 ms，总计 2,020 ms。敌方标题延后至出牌约 2,420 ms 后启动，不再覆盖数值。
- 敌方标题：出牌后约 1,650 ms 出现，HUD 同步切为“敌方回合”；标题完整收束后才让敌人在舞台中现身执行。
- 敌方执行：出牌后约 3,060 ms，敌人前冲、意图闪烁与“敌方执行中”出现；攻击、防护或蓄力约在 3,220 ms 落点。
- 交还玩家：敌方落点后约 640 ms 发牌；HUD 先显示“手牌展开”，玩家章节完整读完后才切为“行动员回合”并重新开放输入。
- 新手牌发入：三张相隔 70 ms，总计约 360 ms。
- Reduced motion：取消飞行、翻转、震动和碎片位移，保留专色切换、数字、评级与声音。
- 回合可读性：战场 HUD 常驻显示“行动员回合 / 敌方回合”，敌方回应在实际伤害、格挡或蓄力前先显示其公开意图和数值；玩家回合章节牌显示“读取意图、选择一张手牌”。胜利与失败均先显示结案章节牌，再进入结果页。

## 强度阶梯

1. 例行：按牌、发牌、普通技术牌。
2. 成功：普通伤害、格挡、恢复。
3. 精准：破解意图、完全格挡、弱点引爆、连续 A/S 评价。
4. 稀有：签名技、击破、强化装订。
5. 危险：玩家受到未格挡伤害、失败。
